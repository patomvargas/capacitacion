Superset: Mejoras y Novedades

2026-03-23 0904 · Superset

Superset es una herramienta de código abierto desarrollada por Apache, que permite a los usuarios crear dashboards personalizados y visualizar datos de manera efectiva. En la versión 6.1.0, se han corregido varios errores, como el problema de la barra de gráficos de ECharts que no podía ordenarse de manera descendente cuando se aplicaba una dimensión.

Para solucionar este problema, se puede utilizar el comando `superset db upgrade` para actualizar la base de datos de Superset. Además, se ha agregado la opción de pre-filtrado para el control de grano de tiempo, lo que permite a los usuarios limitar las opciones disponibles en este control.

La comunidad de Superset ha estado trabajando activamente en la mejora de la herramienta, con 48 colaboradores que han fusionado 330 solicitudes de extracción en febrero de 2026. Esto refleja el ritmo y la dedicación de la comunidad en la mejora de Superset.

En cuanto a la administración de Superset, es importante tener en cuenta la configuración de la base de datos y la seguridad. Se puede utilizar el comando `superset init` para inicializar la base de datos de Superset, y luego configurar la seguridad mediante el archivo `superset_config.py`.

La documentación de Superset también ha sido actualizada, con la adición de nuevos componentes y una guía de diseño de sistema. Esto permite a los desarrolladores crear dashboards y visualizaciones de datos de manera más efectiva y personalizada.
