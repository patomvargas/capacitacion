Mejoras y Novedades en Superset

2026-04-28 1020 · Superset

Superset es una herramienta de visualización de datos que ofrece una variedad de características para crear dashboards y gráficos personalizados. Una de las características más recientes es la adición de una opción de pre-filtro para el control de grano temporal, lo que permite a los usuarios limitar las opciones disponibles en este control. Por ejemplo, se puede utilizar el comando `superset config.py` para configurar las opciones de pre-filtro.

La ordenación descendente en gráficos de barras también ha sido mejorada en Superset. En versiones anteriores, la ordenación descendente no funcionaba correctamente cuando se aplicaba una dimensión, lo que causaba que los datos se ordenaran alfabéticamente en lugar de por el valor total. Sin embargo, este problema ha sido resuelto en versiones más recientes de Superset.

Otra característica interesante en Superset es la capacidad de enviar alertas a servicios de mensajería como Mattermost. Sin embargo, algunos usuarios han reportado problemas con la configuración de las alertas, incluyendo la forma en que se envían los payloads JSON. Para configurar las alertas en Superset, se puede utilizar el comando `superset alert` y especificar el payload JSON utilizando el parámetro `--payload`.

La administración de Superset también ha sido mejorada en versiones recientes. Los administradores pueden utilizar la interfaz de línea de comandos de Superset para realizar tareas como la creación de usuarios y la configuración de permisos. Por ejemplo, se puede utilizar el comando `superset user` para crear un nuevo usuario y asignarle permisos específicos.

En resumen, Superset es una herramienta poderosa para la visualización de datos que ofrece una variedad de características para crear dashboards y gráficos personalizados. Las mejoras y novedades más recientes en Superset incluyen la adición de una opción de pre-filtro para el control de grano temporal, la resolución de problemas con la ordenación descendente en gráficos de barras y la mejora de la administración.

Para obtener más información sobre Superset y sus características, se puede consultar la documentación oficial de Superset y las fuentes proporcionadas a continuación.
