Airflow 3.x: Mejoras y Novedades

2026-03-11 0851 · Airflow

Airflow 3.x es una versión importante en la evolución de este sistema de gestión de flujos de trabajo. Una de las novedades clave es la expansión de la funcionalidad de conciencia de activos y datos, lo que permite a los usuarios tener una visibilidad más detallada de sus flujos de trabajo y datos. Para aprovechar esta funcionalidad, se puede utilizar el comando `airflow db upgrade` para actualizar la base de datos de Airflow.

La integración con otros sistemas y herramientas también ha sido mejorada en Airflow 3.x. Por ejemplo, la integración con Elasticsearch ha sido actualizada para incluir la clase `ElasticsearchRemoteLogIO`, que requiere la versión `apache-airflow-providers-elasticsearch>=6.5.0`. No se necesitan cambios de configuración, ya que las configuraciones existentes como `write_to_es`, `target_index`, `json_format`, `host_field` y `offset_field` siguen funcionando como antes.

Otra novedad importante es la capacidad de crear entornos multi-tenant utilizando grupos de trabajadores compuestos por colecciones de DAG, espacios de nombres de trabajadores dedicados y mapeo de identidad de organización. Esto permite a los usuarios crear entornos de Airflow personalizados y seguros para diferentes organizaciones o departamentos. Para crear un entorno multi-tenant, se puede utilizar el comando `airflow config get` para obtener la configuración actual de Airflow y luego modificarla según sea necesario.

En cuanto a las mejores prácticas, es importante mencionar la importancia de la seguridad en Airflow. Los pequeños errores de configuración o anomalías en las solicitudes pueden parecer inofensivos en sí mismos, pero cuando se combinan, pueden desencadenar incidentes de seguridad. Para evitar esto, es importante monitorear y analizar los registros de Airflow regularmente, utilizando herramientas como `airflow logs` para detectar posibles problemas de seguridad.

Finalmente, es importante destacar la importancia de la comunidad de Airflow y los recursos disponibles para los usuarios. La página oficial de Airflow en GitHub ofrece una variedad de recursos, incluyendo documentación, tutoriales y ejemplos de código. Además, la comunidad de Airflow es muy activa y ofrece una variedad de eventos y webinars para ayudar a los usuarios a mejorar sus habilidades y conocimientos.

En resumen, Airflow 3.x es una versión poderosa y flexible que ofrece una variedad de mejoras y novedades para los usuarios. Al seguir las mejores prácticas y aprovechar al máximo las funcionalidades de Airflow, los usuarios pueden crear flujos de trabajo eficientes y seguros que satisfagan sus necesidades específicas.
