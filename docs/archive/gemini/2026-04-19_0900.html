Gemini: Nuevas Funcionalidades y Mejoras

2026-04-19 0900 · Gemini

La última versión de Gemini introduce la facturación por uso para los modelos Pro y Flash, lo que permite a los desarrolladores pagar solo por lo que utilizan. Además, se han aumentado los límites de velocidad para los modelos Pro y Flash, lo que permite una mayor cantidad de solicitudes por segundo.

Para utilizar la API de Gemini, es necesario crear un proyecto en Google Cloud y habilitar la API de Gemini. Luego, se puede utilizar el comando `gcloud ai gemini init` para inicializar el proyecto y `gcloud ai gemini deploy` para desplegar el modelo.

La documentación oficial de Gemini proporciona una guía práctica para empezar a utilizar la API, incluyendo la elección del modelo adecuado para cada proyecto. Por ejemplo, el modelo `gemini-2.5-flash` es una buena opción para proyectos educativos debido a su equilibrio entre velocidad y costo.

Además, se han lanzado nuevas funcionalidades como Gemini 3.1 Flash Live, que permite a los desarrolladores crear agentes conversacionales en tiempo real. Para utilizar esta funcionalidad, es necesario utilizar el comando `gemini live init` y luego `gemini live deploy`.

La comunidad de desarrolladores de Gemini también ha creado herramientas y tutoriales para ayudar a los desarrolladores a aprovechar al máximo la API. Por ejemplo, el tutorial de Subagentes de Gemini CLI proporciona una guía paso a paso para crear agentes personalizados.
