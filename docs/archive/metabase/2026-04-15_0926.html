Mejoras y Novedades en Metabase

2026-04-15 0926 · Metabase

Metabase es una herramienta de análisis de datos que permite a los usuarios crear consultas, tableros y visualizaciones de datos de manera sencilla. Recientemente, se han incorporado varias mejoras y novedades en el sistema, como la actualización de Netty a la versión 4.2.10.Final para parchear la vulnerabilidad de seguridad CVE-2026-33870, como se indica en `https://github.com/metabase/metabase/issues/71652`.

Además, se ha agregado la capacidad de revertir tablas auto-archivadas después de que regresen, como se describe en `https://github.com/metabase/metabase/issues/72135`. Esto permite a los usuarios recuperar tablas que han sido archivadas automáticamente por Metabase.

Otra característica agregada es la personalización de Metabot por inquilino, que permite a los administradores configurar Metabot de manera personalizada para cada inquilino, como se indica en `https://github.com/metabase/metabase/issues/72116`. Esto es especialmente útil en entornos de multi-inquilino, donde cada inquilino puede tener requisitos y necesidades diferentes.

La transformación de datos entre bases de datos es otra característica agregada, que permite a los usuarios transformar datos entre diferentes bases de datos, como se describe en `https://github.com/metabase/metabase/issues/71238`. Esto es especialmente útil en entornos donde se necesitan combinar datos de diferentes fuentes.

Para configurar estas características, los usuarios pueden utilizar comandos como `git clone https://github.com/metabase/metabase.git` para clonar el repositorio de Metabase y `docker run -p 3000:3000 metabase/metabase` para ejecutar Metabase en un contenedor Docker.

En resumen, Metabase ha incorporado varias mejoras y novedades en su sistema, que permiten a los usuarios tener más control y flexibilidad en el análisis de datos.

Para más información, se puede consultar la documentación oficial de Metabase y las fuentes citadas en este artículo.

Fuentes

  • Bump Netty to >=v4.2.10.Final to patch cve-2026-33870
  • Allow reversing auto-archived tables after they return
  • Per Tenant Metabot customization
  • Cross DB Transforms
